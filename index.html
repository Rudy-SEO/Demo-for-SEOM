How to Implement

To use the optimized CSS in your n8n instance:

  1. Download the optimized-padding.css file
  2. Include it after the original Font Awesome CSS
  3. Apply the utility classes (p-0, p-1, etc.) to elements that need padding adjustments
<!-- In your HTML head -->
<link rel="stylesheet" href="path/to/optimized-padding.css">

<!-- In your HTML body -->
<div class="n8n-card p-2">
  <!-- Your content with reduced padding -->
</div>
